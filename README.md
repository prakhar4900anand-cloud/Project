# Secure Password Generator & Strength Evaluator

## Overview
A Python-based console application that generates strong passwords, evaluates
password strength, and optionally saves passwords for later reference.

## Features
- Generate random passwords with custom length and character options
- Evaluate strength of any password
- Save and view generated passwords from a local file
- Simple text-based user interface

## Technologies / Tools Used
- Python 3.x
- PyCharm (IDE)
- Git & GitHub
- Standard Python libraries: `random`, `string`, `os`, `csv` (or `json`)

## How to Run
```bash
git clone <your_repo_url>
cd password-generator-project
python main.py
